# Support

BBoeOS is a personal hobby project maintained in spare time. There is no formal
support channel and no service-level commitment.

## Where to go

- **Found a bug?** Open an issue at https://github.com/bboe/BBoeOS/issues.
  Please include the QEMU command line, the host OS, and a minimal reproduction.
- **Found a security vulnerability?** Do **not** open a public issue. Use
  GitHub's [private vulnerability
  reporting](https://github.com/bboe/BBoeOS/security/advisories/new) instead.
  See [`SECURITY.md`](SECURITY.md) for scope and expectations.
- **Want to contribute?** Please open an issue to discuss before sending a pull
  request. See [`CONTRIBUTING.md`](CONTRIBUTING.md), which includes the
  copyright-assignment requirement.
- **Just a question?** Issues are fine for now; if the question is more
  conversational, feel free to reach out via the maintainer's GitHub profile.

The maintainer will respond when there is time. If a thread sits idle for a
while, feel free to bump it.
